# Physics Engine Rust-wgpu :


This repo uses wgpu-bootstrap to easily set up the physics engine windows, camera and others : 


### 1. Cube :

```
>> cargo run -p cube
```


### 2. Textured Cube :
```
>> cargo run -p textured_cube
```


### 3. Globe (with Specular Lighting) :

```
>> cargo run -p globe_specular
```
